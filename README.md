# 정보처리기사 & NCS 전산직 퀴즈 웹페이지

📚 **정보처리기사 및 NCS 필기시험 대비**를 위한 HTML 기반 퀴즈 시스템입니다.

## 🚀 주요 기능

- ✅ 문제/보기를 동적 생성 (문제 자동 랜더링)
- ✅ 정답 확인 버튼으로 해설 확인
- ✅ 채점 기능 (총점 표시)
- ✅ 정답일 경우 파란색, 오답일 경우 정답 빨간색 표시
- ✅ 페이지 리프레시 시 문제 순서 랜덤화
